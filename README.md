# Excel Summary API

## Overview
Upload an Excel file and get a summary (sum and average) of specified columns.

## Setup
**1. Install docker-compose**

**2. Clone the repository and navigate to the main folder.**

**3. Build the application with the below command:**

>docker-compose up --build -d


## API Usage
**1. Visit `http://localhost:3000` to use the React interface.**

**2. Choose Excel file from your device.**

**3. Enter column names separated by commas.**

**4. Click "Upload" button and read the result.**


## Warning
In order to use this application in production environment additionals steps should be permormed - adding proper SECRET_KEY and CORS policy.
